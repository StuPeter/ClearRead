# ClearRead - 网页放大镜

🔍 网页文字放大工具，助您清晰阅读，防止被小字忽悠！

## ✨ 主要功能

- **自由缩放**: 支持1-5倍自由缩放，精准查看网页内容
- **大小调节**: 可调节放大镜大小（80-300px），适应不同需求
- **流畅体验**: 高性能渲染，支持页面滚动和动态内容
- **全站适配**: 支持所有网站，无需特殊配置
- **快捷操作**: 丰富的键盘快捷键，提升使用效率

## 🚀 使用方法

### 启动/关闭
- 点击浏览器工具栏的图标启动放大镜
- 再次点击图标或按 `ESC` 键关闭

### 快捷键操作
| 按键 | 功能 |
|------|------|
| `+` / `=` | 放大（每次增加0.5倍） |
| `-` | 缩小（每次减少0.5倍） |
| `[` | 缩小镜头（减少20px） |
| `]` | 放大镜头（增加20px） |
| `R` | 重置所有设置 |
| `ESC` | 关闭放大镜 |

## 🎯 使用场景

- 📖 **阅读长文**: 放大网页文章、小字内容
- 📊 **查看数据**: 放大图表、表格、数字
- 🔍 **细节检查**: 查看图片细节、小按钮文字
- 👁️ **护眼阅读**: 减少眯眼，保护视力
- ⚠️ **防忽悠**: 看清条款、小字说明

## 🎨 界面设计

- **黑色主题**: 专业暗色设计，不干扰阅读

## 📦 安装说明

1. 下载本扩展文件夹
2. 打开浏览器扩展管理页面（chrome://extensions/）
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本文件夹即可安装
